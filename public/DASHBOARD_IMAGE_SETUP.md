# Dashboard Image Setup

Place your real dashboard screenshot at:

```
public/dashboard.png
```

## Requirements:
- Format: PNG or JPG
- Recommended size: 1920x1080 or larger
- Aspect ratio: 16:9 preferred
- Quality: High resolution (Retina-ready)

## The image is used in:
- Home page hero section (DashboardShowcase component)
- Full device/browser frame with animated floating stat cards

## How to replace:
1. Take a screenshot of your TradeInsight dashboard
2. Rename it to `dashboard.png`
3. Place it in the `public/` folder
4. The website will automatically display it
